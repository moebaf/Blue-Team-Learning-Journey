## What is it?
Wireshark is a free, open-source network packet analyser used to capture and inspect traffic in real time. It's one of the most important tools in a blue teamer's toolkit for investigating suspicious activity.

## How it works

1. Select and double-click a network interface (such as Ethernet or Wi-Fi) to begin capturing network traffic.
2. As packets are captured, they are displayed in real time within the packet list pane.
3. Click on any packet to inspect its details, including protocol information, source and destination addresses, ports, and payload data.
4. Customize the interface by adding or removing columns, changing color-coding rules, modifying the layout, and adjusting how packet data is displayed.
5. Apply display filters to focus on specific traffic, such as packets from a particular IP address, protocol, or port.
6. Analyze captured traffic to identify network activity, troubleshoot connectivity issues, or investigate security events.
7. Save captured packets as a .pcap file for later analysis or sharing with other analysts.
