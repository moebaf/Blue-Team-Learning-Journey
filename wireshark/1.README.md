# Wireshark

## What is it?

Wireshark is a free, open-source network packet analyser used to capture 
and inspect traffic in real time. It's one of the most important tools 
in a blue teamer's toolkit for investigating suspicious activity.

## Why it matters for Blue Team

- Detect unusual traffic patterns

-  Investigate potential intrusions

- Analyse malware communication
  
- Reconstruct attack timelines from pcap files
