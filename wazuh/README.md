## What is it?

Wazuh is a free, open-source security platform that combines Host-Based Intrusion Detection (HIDS), Security Information and Event Management (SIEM),
log analysis, file integrity monitoring, vulnerability detection, and incident response capabilities. It collects and analyzes security data from endpoints,
servers, cloud environments, and network devices to help detect and respond to security threats.

## Why it Matters for Blue Team

- Centralize and analyze security logs from multiple systems
- Detect suspicious activities and potential security incidents
- Monitor file integrity and unauthorized system changes
- Identify vulnerabilities and security misconfigurations
- Support threat detection, incident response, and threat hunting
- Improve visibility across endpoints, servers, and cloud environments
- Automate alerting and event correlation
- Assist with security compliance and auditing requirements
