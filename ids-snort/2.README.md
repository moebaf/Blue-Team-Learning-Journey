# Snort IDS

## What is it?
Snort is a free, open-source Intrusion Detection System (IDS) and Intrusion Prevention System (IPS) that monitors network traffic and generates alerts 
when it detects suspicious or malicious activity based on predefined rules and signatures.

## Why it matters for Blue Team
- Detect network-based attacks in real time
- Identify known threats using signature-based detection
- Generate security alerts for suspicious activity
- Support incident response and threat hunting efforts
- Monitor network traffic for policy violations and malicious behavior
- Help validate security controls and network visibility
